# 🚀WT32-ETH01_Net

## 網路設定與連線

## 📌系統設定步驟
### 1. 初始設定模式
- 當WT32-ETH01首次啟動或重置時：
- 建立WiFi AP模式（SSID: WT32-ETH01_Config）
- 預設IP: 192.168.4.1
- 用戶連接到此AP並訪問配置頁面

### 2. 正常操作模式
- 根據存儲的配置連接WiFi或以太網
- Web服務器在連接的網絡上運行
- 可通過分配的IP地址訪問配置頁面

### 3. 配置恢復
- 如果無法連接任何網絡：
- 長按復位按鈕10秒恢復AP模式
- 或通過串口重置配置

## 📌部署流程
### 1. 編譯和上傳固件
- 安裝Arduino IDE或PlatformIO
- 安裝必要的庫：WiFi、ETH、WebServer、ArduinoJson、Preferences
- 編譯並上傳代碼到WT32-ETH01

### 2. 文件系統上傳
- 使用SPIFFS工具上傳前端文件
- 確保index.html在SPIFFS根目錄

### 3. 使用流程
- 首次啟動時連接WT32-ETH01_Config WiFi
- 瀏覽器訪問192.168.4.1
- 掃描並配置WiFi或以太網
- 系統將重啟並使用新配置連接



## 📌安全性考慮
1. 配置AP密碼保護
2. HTTPS支持（需額外證書）
3. API認證令牌
4. 配置存儲加密

## 📌故障排除
### 1.無法訪問Web界面
- 檢查網路連線
- 確認IP地址
- 重啟設備

### 2.WiFi掃描無結果
- 確保WiFi天線正常
- 檢查周圍WiFi信號

### 3.以太網無法連接
- 檢查網線和路由器
- 確認LAN8720電路正常

此架構提供了完整的Web配置界面，使用者可以方便地管理WT32-ETH01的網路連接，支援WiFi和乙太網路的無縫切換。
